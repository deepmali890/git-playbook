## 🛠 Common Git Commands

### Initialize Repository

```bash
git init
```

### Check Status

```bash
git status
```

### Stage Files

```bash
git add .
```

### Commit Changes

```bash
git commit -m "Your commit message"
```

### Push Changes

```bash
git push origin main
```

### Pull Changes

```bash
git pull origin main
```
