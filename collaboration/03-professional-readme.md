# GitHub Pages

## What is GitHub Pages?

GitHub Pages is a free hosting service provided by GitHub for static websites.

### Supported Files

* HTML
* CSS
* JavaScript
* Images

---

## Use Cases

* Personal Portfolio
* Project Documentation
* Landing Pages
* Blogs
* Resume Websites

---

## Steps to Host a Website

### 1. Create a Repository

Example:

```text
my-portfolio
```

### 2. Upload Website Files

Example:

```text
index.html
style.css
script.js
```

### 3. Enable GitHub Pages

```text
Repository
    ↓
Settings
    ↓
Pages
    ↓
Select Branch (main)
    ↓
Save
```

### 4. Website Goes Live

Example URL:

```text
https://username.github.io/repository-name/
```

---

## Main Website File

```text
index.html
```

GitHub Pages looks for `index.html` as the entry point.

---

## Custom Domain (Optional)

Example:

```text
www.dileepmali.com
```

Can be connected to GitHub Pages using DNS settings.
